This index contains entry points to more information about PC Native Oberon. In order to keep it as up-to-date as possible, the information is stored on the World-wide Web.
  • PC Native Oberon home page. This is the official PC Native Oberon home page.
  • Technical information (also available locally). Information about traps, the SYSTEM module for low-level programming, device drivers, low-level tracing, boot diskettes, configuration strings, object file format, heap block format etc.
  • Release history (also available locally). Detailed information about updates in the various releases, as well as the hardware requirements of the system.
For users without WWW access, a "snapshot" of some of the web pages are available locally as part of the Documentation package. Some links on the local pages lead to WWW pages. If an Internet connection is not available, press ESC to abort loading of these pages.
